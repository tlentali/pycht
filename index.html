    üé® Pycht

    Pycht is a lightweight Python tool that transforms images into colorful street art-style stencils using K-Means clustering.

    It automatically reduces an image‚Äôs color palette into distinct clusters and generates transparent PNG layers for each one ‚Äî ideal for digital or physical stencil creation, creative coding projects, or simply exploring image segmentation.

    ‚ú® Features

    • üß† Simple image clustering using Scikit-Learn‚Äôs K-Means algorithm
    • üñºÔ∏è Color separation with transparency masks
    • üìÅ Input/output file handling with minimal setup
    • üß∞ Modular architecture for easy extension

    üöÄ How It Works

    1. Load an image.
    2. Process the pixels into a 2D format.
    3. Cluster colors using K-Means.
    4. Separate each color cluster into its own transparent stencil.
    5. Save the final image and each stencil layer as a .png.

    üì¶ Example Usage

    import pycht

# Create a stencil with 5 color clusters
pycht.stencil("images/input.jpg", nb_colors=5)

    This will generate: - stencil_final.png ‚Üí final clustered image - stencil_1.png, stencil_2.png, etc. ‚Üí one per color cluster, with transparency
